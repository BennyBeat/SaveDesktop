Saving Configuration

SaveDesktop allows you to save the current configuration of your Linux desktop environment. Simply, enter a name for the configuration file, select the items you want to save and click the "Save" button.

Periodic saving

If you want SaveDesktop to save your environment settings for you, no problem. Simply choose whether you want to save your configuration daily, weekly, or monthly. You can also choose a folder for regular saving, for example to an external drive or the cloud, and be sure you won't lose your settings again.

Importing configuration

Because the configuration files are saved in Tar Gzip format, you can easily see what configuration files are in the file and "tweak" them if necessary. Importing configuration in SaveDesktop is really a breeze. Simply switch to the Import tab and click the "Import from File" button, and once the configuration import is complete, you can log out of the application directly to make the changes take effect.

Synchronization configurations

If you own multiple computers, SaveDesktop won't let you down either. With synchronization, you can keep the systems of two or more computers unified. To do this, just switch to the Sync tab in the app. On the first computer, click Set up the sync file, then select the periodic save file, the file where your settings are saved using the periodic save feature, and then just choose how often you'd like the sync to run or whether you want to do it manually from the app's interface. Then just paste the URL to sync to the other computer and you're ready to start syncing!
